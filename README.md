# ProgramaOrdenesPescaderia
Un software para celular hecho en C# Maui
Pensado para aplicaciones moviles pensando para el mesero donde podra realizar ordenes, visualizar las mesas activas, ver el total y subtotal de cada mesa, ver el historial de todas las mesas atendidas a lo largo del turno, agregar, eliminar y editar platillos del menu, y mas funciones por haber.

Por el momento se cuenta con 2 versiones, uno mas sencillo donde solo se puede ordenar, ver las mesas y el historial.
El otro aun esta en progreso de desarrollo donde tenga un CRUD y genere un archivo json donde guarde y lea todos los nuevos platillos anadidos por el usuario.
